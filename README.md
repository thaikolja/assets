# What are you doing here?!

Well, well, well, look at you! Looks like your curiosity brought you to a repository you've absolutely no business being in. Shame on you!

No, seriously. This repository is an **asset repository**, meaning I use it to store assets online, such as images or screenshots that must stay in `README.md` files. I use it for both Git platforms, [on GitHub](https://github.com/thaikolja) and [on GitLab,](https://gitlab.com/thaikolja) to avoid paying AWS or other mega-corporations money.

## File Storage

Most files stored on here will be either image or video files that are public anyway, just in case you were about to waste your time finding out anything valuable.

My favorite Markdown editor, **[Typora](https://typora.io/)**, has a great feature that lets me upload any image or video files I drag and drop into the editor. It gives me a few slightly sketchy-looking Chinese providers, along with the option to run a command to do this and return the URL.
